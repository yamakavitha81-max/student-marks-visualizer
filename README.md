# Student Marks Visualizer 📊

## Description

A Python-based Student Marks Visualizer that allows users to enter student names and marks, then display the data using different types of charts. The project uses Matplotlib to create visual representations of student performance through a simple menu-driven interface.

## Features

* Enter student names and marks
* Display marks using a Bar Chart
* Display marks using a Line Chart
* Display marks using a Pie Chart
* Menu-driven program
* Beginner-friendly Python project

## Technologies Used

* Python 3
* Matplotlib

## How to Run

```bash
python student_marks_visualizer.py
```
